## Instalación

Para configurar el proyecto de manera local, realizar los siguientes pasos.

### Pre-requisitos

* npm
  ```sh
  npm install npm@latest -g
  ```

### Instalación



1. Clonar el repositorio
   ```sh
   git clone https://github.com/largat12/curso-react-js.git
   ```
2. Instalar paquetesNPM
   ```sh
   npm install
   ```
